# How to add a new product to Pablo IA

1. Run intake protocol (agents/core/intake.md)
2. Create [product]-dev repo under your GitHub account
3. If needs conversational AI interface → use an agent-os shell pattern
   Define the product brain in [product]-brain-dev or in the product repo's CLAUDE.md
4. Add to PROJECTS.md index
5. Create projects/dev/[product].md
6. Create empty projects/uat/[product].md and projects/prod/[product].md
7. Add to portfolio/README.md when ready to showcase
8. Register relevant domain agent if specialized knowledge needed
